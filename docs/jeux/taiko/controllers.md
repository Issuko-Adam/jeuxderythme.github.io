# Quel controller pour jouer à Taiko No Tatsujin?

## `Hori: Taiko Drum Controller For PS5`

**Prix :** 90€ (Neuf, France)

**Lien :** [Site Officiel d'HORI](https://horieurope.com/fr-eu/products/spf-042e)

**Compatibilité :** PS5, PS4, Windows 10/11<br><br>*Il existe aussi une version pour XBOX ainsi qu'une version switch plus répendue dans le marché d'occasion*

### Commentaire / Retours

> ℹ️ **À noter**
>
> Nous parlerons de la version PS5 de ce controller dans ce guide, car il est un peu plus sensible en sortant d'usine que ses prédécesseurs. Mais toutes les informations mentionnées sont vraies pour toutes les itérations de ce controller.

Il s'agit du controller basique pour une première expérience à la maison. C'est notamment le tambour que vous trouverez dans les différents bundles sur Nintendo Switch.


Les tambours HORI ne sont vraiment sensibles quand ils sont neufs, beaucoup de vos inputs ne seront pas enregistrés. C'est pour cela que beaucoup de joueurs [modifient](https://www.wkd4496.net/tuto-taiko-no-tatsujin-hori-ps4-095-taiko-drum-controller-mod/) leur tambour pour avoir une expérience de jeu convenable.

Contrairement aux tambours sur borne d'arcade qui ont une surface en caoutchouc sur une plaque de bois, il s'agit ici d'un tambour en plastique, avec des baguettes qui sont elles aussi en plastiques. Le ressenti comme les rebonds ou le bruit seront donc très différents et ne représenteront pas l'expérience que vous pouvez retrouver en salle d'arcade.

Nous recommandons ce tambour pour une première expérience seulement si vous arrivez à le trouver d'occasion pour un prix raisonnable. Si vous comptez débourser la somme neuf, Il est préférable de mettre une dizaine d'euros de plus pour le [TDC10 de chez rythmagica](https://www.rythmagica.net/products/pro-taiko-drum-controller).
